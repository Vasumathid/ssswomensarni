GOOGLE ANALYTICS (to track visitors properly):
------------------------------------------------
1. Go to analytics.google.com
2. Create account → Add property → Website
3. You get a code like: G-XXXXXXXXXX
4. Open index.html, about.html, academics.html, campus.html, contact.html
5. Find: GA_MEASUREMENT_ID
6. Replace with your actual code (e.g. G-ABC123XYZ)
7. Now Google tracks all visitors in real detail
